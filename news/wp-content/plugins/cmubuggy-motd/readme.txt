=== CMUBuggy MOTD Controller ===
Requires at least: 6.8
Requires PHP: 7.4
Tested up to: 6.8.4

=== Description ===
A very simple way to leverage the wordpress authenticaltion mecahnism to allow us to post a short
MOTD at the top of cmubuggy.org.

No purpose outside of cmubuggy.org.

=== Installation ===
Should wind up in the correct place when you sync the cmubuggy.org git repository.

Make sure to activate it.
